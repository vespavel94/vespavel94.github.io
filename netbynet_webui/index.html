    Настройки соединения:

  • Имя сети – название вашей сети
  • Метод шифрования – выберете метод шифрования (по умолчанию WPA2-PSK, наиболее защищенный метод шифрования)
  • Пароль – создайте ваш пароль не менее 8 символов
  • При создании, рекомендуется использовать сложные цифро-буквенные пароли для того, чтобы максимально усложнить задачу его подбора.
  • Частотный канал – выберете канал из списка, либо используйте «автовыбор». Выбор определенного канала для диапазона 2.4 ГГц может улучшить качество соединения
  • После того, как завершите настройку, обязательно нажмите «Применить».
Порты для IPTV

Без данной настройки работа IPTV приставки может быть некорректна, т.к. для правильной работы необходимо, чтобы приставка получала прямое соединение с сервером интернет провайдера.

Проставляя знак выбора напротив порта, вы выделяете данный порт для подключения IPTV STB приставки, подключенной по кабелю к выбранному порту роутера.

Будьте внимательны при выборе порта для IPTV, т.к. если вы укажете порт неверно, может прекратиться выход в сеть для устройств, подключенных к данному порту, если они не являются IPTV приставкой.

USB

Подключите к USB порту роутера flash накопитель объемом не менее 128 мегабайт, на котором должно быть свободно не менее 50 мегабайт дискового пространства. Данное дисковое пространство необходимо для корректного сохранения на flash накопитель данных.

    Опции:

  • DLNA – применяется, если вы подключаете flash накопитель с каким-либо медиа контентом (музыка, фильмы и т.д.), поддерживающим обращение по стандарту DLNA, и желаете просматривать это на телевизоре, или компьютере, подключенном к роутеру.
  • SAMBA – используется, если вы желаете использовать роутер как контроллер домена, в этом случае по USB должен быть подключен накопитель с соответствующим пакетом программ.
  • FTP – используется, если вы желаете использовать подключенное устройство как FTP сервер для обмена файлами и данными. После выбора пункта FTP можно будет изменить настройки приватности.
Интернет

PPPoe

«PPPoe» - это метод подключения к сети, использующий обязательную пару логин и пароль для соединения с интернетом.

    Для настройки соединения укажите в полях:

  • Логин – ваш логин
  • Пароль – ваш пароль
  • Информация для заполнения полей есть в памятке для абонента.
  • После того, как завершите настройку, обязательно нажмите «Применить».

IPoe (Динамический IP)

«IPoe (Динамический IP)» - то метод подключения к интернету, при котором все настройки для выхода в интернет, оборудование получает автоматически от сети интернет провайдера.

Для настройки соединения не требуется каких-либо дополнительных настроек.

    Вы можете изменить используемые DNS сервера, поставив флаг, напротив пункта «Настроить вручную», и затем вписать в поля:

  • DNS1 – основной DNS сервер
  • DNS2 – вторичный DNS сервер

Вы можете изменить используемый MAC адрес, либо прописав его вручную в формате XX:XX:XX:XX:XX:XX, либо нажатием на «Клонировать MAC адрес» прописать такой же MAC адрес, как у устройства, с которого осуществлено подключение к оборудованию.

После того, как завершите настройку, обязательно нажмите «Применить».

IPoe (Статический IP)

IPoe (Статический IP) - это метод подключения к интернету, при котором настройки соединения с сетью интернет провайдера должны быть прописаны вручную в полях для заполнения.

    Для настройки соединения укажите в полях:

  • Маска подсети – маску в формате ххх.ххх.ххх.ххх, либо в формате /хх
  • Шлюз по умолчанию – шлюз в формате ххх.ххх.ххх.ххх
  • DNS1 – основной DNS сервер провайдера
  • DNS2 – вторичный DNS сервер провайдера
  • MAC адрес – MAC адрес в формате XX:XX:XX:XX:XX:XX
  • Информация для заполнения полей есть в памятке для абонента.
  • После того, как завершите настройку, обязательно нажмите «Применить».
